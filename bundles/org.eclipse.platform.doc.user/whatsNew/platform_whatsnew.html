Views, Dialogs and Toolbar

Open Resource dialog now defaults to "substring" search By default, the search pattern input to the Open Resource dialog is now searched anywhere in the file name. That means you no longer need to input a leading asterisk ("*") when searching from the middle of a file name.

If you want to enforce the old behavior, where "prefix-matching" search is expected, simply add a leading angle bracket (">") to the search pattern.
